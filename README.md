# Quant Fundamental ML

Interpretable machine-learning-enhanced fundamental quant strategy.

## Objective

This project studies whether a fundamental quant strategy enhanced with interpretable machine learning can improve risk-adjusted performance compared with passive buy-and-hold benchmarks.

## Strategy Logic

The model is not used as a price oracle.

The pipeline is:

1. Data ingestion
2. Point-in-time data cleaning
3. Feature engineering
4. Fundamental ranking
5. Label construction
6. Interpretable ML model training
7. Signal generation
8. Portfolio construction
9. Walk-forward backtest
10. Benchmark comparison
11. Reporting

## Disclaimer

This repository is for educational and research purposes only. It is not financial advice.
